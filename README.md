# Application Security Lab  OWASP Top 10

Hands-on notes and reproduction steps for OWASP Top 10 vulnerabilities in a controlled lab environment (DVWA  Damn Vulnerable Web Application).

## Covered So Far

| # | Vulnerability | Status |
|---|---|---|
| A01 | Broken Access Control | Done |
| A03 | Injection (SQL) | Done |

## Environment
- DVWA running on Ubuntu VM (VirtualBox)
- Browser: Firefox with Burp Suite (community edition) as proxy for request inspection
- DVWA security level: Low (learning phase)

## Objective
Understand how each vulnerability works from an attacker perspective before studying the defence/remediation side.
